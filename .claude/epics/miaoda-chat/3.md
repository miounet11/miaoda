## Overview
Implement a comprehensive search and filtering system for MiaoDa Chat that allows users to efficiently find conversations and messages across their entire chat history. This includes advanced search UI, backend optimization, saved search queries, and intelligent relevance scoring.

## Problem Statement
Users need powerful search capabilities to navigate large conversation histories effectively. Current search functionality may be limited and doesn't provide the granular filtering and organization features needed for productive chat management.

## Acceptance Criteria
- [ ] Implement advanced search UI with multiple filter options
- [ ] Add full-text search across message content with relevance scoring
- [ ] Support filtering by date ranges, participants, and message types
- [ ] Implement saved search queries and search history
- [ ] Add real-time search suggestions and autocomplete
- [ ] Optimize search performance for large datasets
- [ ] Support regex and boolean search operators
- [ ] Implement search result highlighting and context preview
- [ ] Add export functionality for search results
- [ ] Ensure search results are paginated and performant

## Technical Details

### Search Interface Components
- Enhanced search bar with advanced options toggle
- Filter sidebar with date pickers, participant selection
- Search results view with message context and navigation
- Saved searches management interface
- Search history and suggestions dropdown

### Backend Search Implementation
- Leverage optimized FTS indexes from Task 001
- Implement search result ranking and relevance scoring
- Add search caching and result pagination
- Support for complex query parsing and execution

### Search Features
- **Content Search**: Full-text search across message bodies
- **Metadata Filters**: By date, participant, chat type, attachments
- **Advanced Operators**: Boolean AND/OR, phrase matching, wildcards
- **Contextual Results**: Show surrounding messages for context
- **Export Options**: Save search results to various formats

- `src/main/db/search.ts` - Backend search implementation

- Search should feel instantaneous for common queries
- Progressive disclosure of advanced search options

## Performance Requirements

- Real-time search suggestions with debounced input

- Memory-efficient search result caching

## Risks & Mitigation
- **Risk**: Search performance degrades with very large datasets

- **Risk**: Complex search queries may confuse users
